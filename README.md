
Android Architecture Components
===============================

A collection of samples using the [Architecture Components](https://developer.android.com/arch):

- Lifecycle-aware components
- ViewModels
- LiveData
- Paging (preview)


A few other libraries

- Realm
- Retrofit
- RxJava
- Dagger
